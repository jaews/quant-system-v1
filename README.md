# Quant System V1

Minimal starter quant trading system project.

Quick start:

1. Create and activate a Python virtual environment (Windows PowerShell):

```powershell
python -m venv venv
venv\Scripts\Activate.ps1  # may require execution policy change
```

Or use cmd:

```cmd
venv\Scripts\activate.bat
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the simple main runner:

```bash
python -m src.main
```

4. Run tests:

```bash
pytest -q
```
