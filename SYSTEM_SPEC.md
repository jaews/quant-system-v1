# System Specification

This repository contains a minimal quant trading system skeleton.

Components
- `src/data.py` - sample data loader
- `src/signals.py` - signal generation (SMA crossover)
- `src/backtest.py` - simple backtest engine
- `src/portfolio.py` - position sizing helpers
- `src/risk.py` - risk metrics (max drawdown)
- `src/report.py` - simple reporting utilities
- `src/main.py` - runnable entry point

Non-functional requirements
- Python 3.8+
- Reproducible sample data for tests
